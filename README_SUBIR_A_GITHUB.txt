Cómo subir:
1. Descomprimí este ZIP.
2. Subí todos los archivos al repositorio Verduraps.
3. Reemplazá los archivos existentes.
4. Abrí https://verduraps.github.io/Verduraps/

Si la tablet sigue mostrando la versión vieja:
- Desinstalá la PWA anterior o borrá datos del sitio en Chrome.
- Volvé a abrir desde Chrome e instalá nuevamente.
